Crear página web usando GitHub.io

Objetivo:

Crear un repositorio conteniendo una página web y utilizar GitHub.io para automatizar la creación de la página con el contenido del repositorio.

Información del Proyecto

  • Repositorio GitHub: `.github.io/`
  • GitHub Pages URL: `https://.github.io//`

Tecnologías Utilizadas

  • HTML5
  • CSS3
  • GitHub Pages
  • Git para control de versiones

Proceso de Desarrollo

  • Creación del repositorio en GitHub
  • Clonación del repositorio localmente
  • Desarrollo de la página web (index.html)
  • Configuración de GitHub Pages
  • Despliegue automático
